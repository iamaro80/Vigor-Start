create folder and add three files at least (*.py files in which has the functions 
                                            or any code, setup.py, README.txt)
# use the setuptools

1. Create a distribution description
	setup.py	
		----
			from setuptools import setup

			setup(
				name = 'vsearch',
				version = '1.0',
				description = 'The Head First Python Search Tools',
				author = 'HF Python 2e',
				author_email = 'hfpy@gmail.com',
				url = 'headfirstlab.com',
				py_modules = ['vsearch'], # a list of files to include
				
			)
		----
	README.txt
		----
			any description about the module
		----
2. Generate a distribution file
	[folder path where all ]\py -3 setup.py sdist

3. Install the distribution file
	py -3 -m pip install vsearch-1.0.zip # or vsearch-1.0.tar.gz
